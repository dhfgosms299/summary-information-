{position : 속성;}

static - 요소를 문서의 흐름에 맞추어 배치
( 좌표를 기입하지 않는다 ).
relative - 이전 요소에 자연스럽게 연결해 배치 위치를 지정할수있음
absolute - 원하는 위치를 지정 배치
fixed - 지정한 위치에 고정 배치. 화면에서 요소 잘릴수있음


position - relative
relative = 상대위치 기준 - 앞에있는 요소
- 자연스럽게 배치
- 고정되어 있지 않고 - 다름요소에 의해 바뀔수있음
- 상대적인 위치를 사용하기 때문 다름요소와 조화를 이룬다
- left 나 top 속성을 이용해 요소의 위치를 옮길수있다.


absolute 속성
센터 위치는 표시되는 도형의 반지름의 크기만큼을 좌표에서 빼주면 중앙의 위치에
도형을 놓을수있다.
- 문서의 흐름과는 상관없이 원하는 위치에 배치
-요소의 위치는 가장 가까운 부모 요소나 조상 요소중 position : relative 인요소
- left, right , top ,bottom 속성을 사용해 네 모서리에 얼마나 떨어져 있는지 지정
상위요소가 relative,absolute 인 경우에 absolute 속성을
사용하여 기준위치로 사용할수 있고, absolute 속성을 사용한 박스는 relative 속성을
사용할 경우 다르게 위치가 바뀐다.
relative 기준으로 사용하고 absolute 는 실제 박스 이동하여
표시할때 사용 한다.


position : fixed 속성
- 문서의 흐름과는 상관없이 원하는 위치에 요소 배치
- 부모의 요소가 아닌 브라우저 창 기준
브라우저 창 위쪽 위 꼭지점 (0.0) 기준으로 좌표 계산
- 브라우저 창 화면을 스크롤하더라도 계속 같은 위치 고정.

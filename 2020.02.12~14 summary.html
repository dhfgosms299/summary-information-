     가로방향 : 행 < row = 줄

     세로방향 : 열 < columm = 칸

     표만드는 법 행 : tr / tr

     열 : td / td

     < table / table 다음 < tr / tr 다음 < td / td (table low 위아래 열) (table data 왼쪽 오른쪽 행)

     width (폭) : 값 (너비 값) table width="숫자 값" < td / td 에 도 적용 가능

     테두리 table border="숫자" (선의 굵기) 숫자의 수치가 커지면 커질수록 굵기가 두꺼워짐

     tr border="숫자"  tr 태그 에서는 선이 안들어감

     td border="숫자"  td 태그 에서도 선이 안들어감

     Table =width="100%" or ="속성값" 로 나눌시에는 테이블을 100% 지정하고 < td / td 값 에서 나눠야함

     또는 < td / td 값으로 폭을 조정할때 전체를 100 퍼센트로 나눠서 각 배분해야함

    표 칸 합치는 법
    

    

    왼쪽 오른쪽 으로 합칠시 (칸을 합칠시) colspan="합치고싶은 (갯수) 칸수"
    

    

    줄을 합칠시 (줄) 밑줄 방향은 밑이나 위쪽 rowspan="합치고싶은 (갯수) 줄수"
    

     
    합친칸은 반드시 지워야함 (ex  (테이블 변형) 지운 숫자 (6,9,13,14,17,18,19,20)
    

    

    

    colspan 으로 왼쪽이나 오른 쪽을 합치고 rowspan 으로 그 합친 칸을 합칠수있다

    단 주의사항으로 같은 조건이여야함

    (ex (테이블 변형) colspan (2,3) + colspan (7,8) + rowspan="합치고싶은 칸수")

     그리고 그다음에 마지막으로 수정한 colspan 지정한 위치를 지운다

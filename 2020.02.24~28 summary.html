{background-clip:(밑에 코드);} 속성

       border-box - 박스 모델의 가장 외곽 테두리 까지 적용 (기본 값)
       padding-box - 박스 에서 테두리를 뺀 패딩 (padding) 범위 까지 적용

{background-image: url(파일경로 또는 사이트 이미지 링크);} 속성

          background-image 는 전체를 체우는 기능이므로 사진 크기에 따라
          사진이 작으면 바둑 판식으로 사진이 나옴 div 로 도 이미지 삽입 가능
      
{background-repeat: (속성);}

          repeat - 브라우저 화면에 가득찰 때까지 배경 이미지를 가로 세로 반복
          repeat-x - 가로로 반복
          repeat-y - 세로로 반복
          no-repeat - 배경 이미지를 한번만 표시하고 반복 없음
          {background-repeat:repeat;} 기본값으로 생략 가능
          {background-repeat:repeat-x;} 가로로만 한줄로 표시
          {background-repeat:repeat-y;} 세로로만 한줄로 표시
          {background-repeat:no-repeat;} 이미지를 한번만 표시
          {background-repeat:round 와 space;} 는 이미지가 짤리지 않고 채워지도록 조정한다.
      
{background-size: (속성);}

         background-size 는 주로 글상자에 적용하여 사용한다.
         body 에는 화면 크기 맞는 사이즈를 no-repeat 속성값으로
         주로 사용한다

          auto - 원래 배경 이미지 크기 로 표시

          contain - 요소안에 배경 이미지가 다 들어오도록 이미지 확대/축소
                    body 배경 사이즈를 설정할때  contain 속성은세로길이로 맞춤
                    높이가 지정 되어야 제대로 표시함

          cover - 배경 이미지로 요소를 모두 덮도록 이미지 확대/축소 폭 으로 맞춤

          <크기값> - 너비 높이 값지정 너비 값만 지정할 경우 

                     원래 배경 이미지 크기를 기준으로
                     축소/확대 비율 자동 계산해 높이 값 지정

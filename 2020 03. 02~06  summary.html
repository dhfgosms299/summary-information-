liear-gradient - 그라데이션 효과 And 각도 효과 (deg = deg)

   gradient 속성은 background 속성을 사용함
   Gridation 속성은 body 부분에서는 잘사용 안함
   선형 그라데이션 색상이 (수직또는 수평 또는 대각선 방향으로 일정하게 변하는것)
   {linear-gradient(각도 or  방향,색상);}
   Direction (to top, to left, to right, to bottom)
   (ex
   body{height:1000px;
   background:linear-gradient(to top or 45deg 각도,white,blue,red,yellow);}
   색상은 오른쪽에서 부터 순서대로 지정됨 white 색깔이 마지막
   각도 (degree 속성) - 코드로 deg
   
원형 그라데이션 - radial-gradient

   원의 중심과 크기를 지정해 그라데이션 모양을 바꿀수있음
   기본형 {radial-gradient(circle,색상);} 색상은 가운데부터 바깥쪽으로
   모양을 표시하지않으면 ellips 타원형 이 기본으로 표시된다
   (ex
   body{height:1000px;
        background:radial-gradient(circle,white,yellow,red);}

         %는 0 - 50 지정한다

(ex < style >
    div {width : 200px;
         height : 200px;
         border : 3px solid black;
         background-color : gray;
         background-clip :content-box;
         padding : 10px;
         margin : 50px;
      }
    .box1 {border-top-left-radius: 10px;
           border-top-right-radius: 20px;
           border-bottom-left-radius: 30px;
           border-bottom-right-radius: 40px;}
    < /style >
     border-radius 생략 하는법 
    (ex

    < style >
       div{ width : 200px;
            height : 200px;
            border : 3px solid black;

       .box1{border-radius : 10px 20px 30px 40px;}
              from top-left to bottom-left
    위치를 생략하면 top-left 부터 시계방향으로 표시된다
       .box2{border-radius : 10px 40px;}
           두개로 표현하면 왼쪽 위 대각선 방향으로 같은 값으로 표시된다
        .box3{border-radius : 30px;}
           하나로 표시하면 전체 값으로 표시됨
   < /style >
   

   

   
box-shadow 속성

   <수평 거리 x> - 양수 겂은 요소의 오른쪽, 음수 값 요소의 왼쪽

   <수직 거리 y> - 양수 값은 요소의 아래쪽, 음수 겂 요소의 위쪽

   <흐림정도> - 0을 기본 값으로 해 진한 그림자 표시

   <번짐정도> - 양수값을 사용하면 그림자가 모든 방향으로 퍼져
                         그림자가 박스보다 크게 표시
               음수값 그림자가 모든 방향으로 축소되어 표시 기본값 0

   <색상> - 그림자의 색상지정 한가지만 지정 가능 , 여러개의 색상 지정

   < inset > - 안쪽 그림자 설정 필요한 경우에만 사용하는 옵션 값

   (ex

   < title >box-shadow 속성 < /title >
    < style >
       div { width : 100px;
             height : 100px;
             border : 1px solid pink;
             color : orange;
             margin : 50px;}
       .box2{box-shadow : 5px(x) 5px(y) 0px(흐림) 10px(번짐) gray;}
       .box3{box-shadow : 5px 5px 5px 0px gray;}
       .box4{box-shadow : 5px 5px 5px 0px gray inset(안쪽에 그림자);}

    < /style >
    

    

  
margin 속성


  <크기> - 너비나 높이 값 px(픽셀) 이나 cm(센티미터) 같은 단위와 함께 수치로 지정

  <백분율> - 박스 모델을 포함하고 있는 부모 요소 기준 너비나 높이 값 % 지정
           ex- margin : 0.1%;
  auto - display 속성에서 지정한 값에 맞게 적절한 값 자동지정

  margin 이 겹치는 경우에는 마진 값이 큰쪽으로 겹쳐진다.
         (세로 margin OK /  가로 margin 은 일정한 간격 X)

 CSS 레이아웃 (positioning) 

   
{box-sizing : (속성);} - 박스 너비 기준 정하기 (p328)

   content-box - width 속성 값을 콘텐츠 영역 너비 값 사용. (기본값)
   border-box - width 속성 값을 콘텐츠 영역 테두리까지 포함한 박스
                모델 전체 너비 값.
  

  
{float : (속성);} - 왼쪽이나 오른쪽으로 배치하기

   left = 해당요소를 문서의 왼쪽으로 배치
   right = 해당요소를 문서의 오른쪽으로 배치
   none = 좌우 어느쪽도 배치하지 않음.
   {clear : both;} - 속성 사용시 float 속성을 해제할수있음
   display vs float

    display - 속성은 문제가 발생 (정렬 X) 순서를 못바꿈 (배치정렬 X)
         (inline - div 태그 정렬 X 그래서 inline-block 속성을 사용해야함)

   float - display 속성에 비해 오른쪽 왼쪽 정렬이 가능함
             (배치정렬 가능)  가운데 정렬이 안됨.
             float 속성을 없앨시 반드시 {clear : (both,left,right);}
